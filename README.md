# Institute of Digital Risk Homepage

This is a simple homepage for the Institute of Digital Risk.

## Logo Design Note

The logo features a geometric cube shape to symbolize structure, risk assessment, and resilience in digital environments. The primary color orange (#FF6600) represents innovation and energy, while black provides strong contrast and professionalism, with white for clean backgrounds. Typography uses a clean, minimalist sans-serif font (Arial) for a tech/education feel, ensuring high legibility at small sizes for favicons and mobile headers.

## How to run

Open index.html in a web browser.
